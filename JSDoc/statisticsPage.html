statisticsPage

This file contains the code and functions which control the statistics page. Data is retrieved from the storage, and is passed to various tables.

Methods

(static) connectHtmlFunctionality()

Connects HTML functionality to javascript classes

(static) initStatisticsPage()

Initialize HTML elements and set the data in the tables. Retrieves the data from the storage using Promises. Waits until all data is retrieved, and then populates the interceptionCounterTable, the blacklistTable and the exerciseTimeTable.
